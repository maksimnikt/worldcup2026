⚽ ЧМ‑2026 Группы → Полная сетка плей‑офф

Задайте очки (Pts). При равенстве очков в группах вы можете вручную расставить места с помощью кнопок ↑ и ↓.
В плей‑офф выходят: 1‑е и 2‑е места из каждой группы + 8 лучших 3‑х мест (вы можете менять порядок внутри каждой группы с одинаковыми очками).
После расстановки нажмите «Зафиксировать места» – сетка построится автоматически по официальной таблице Annexe C.

📋 Группы

🏆 Плей‑офф (клик по команде = победитель / снять)

Сетка построена по официальному расписанию ЧМ‑2026. Все раунды до финала.
Зелёным выделены победители матчей.
